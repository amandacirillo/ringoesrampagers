PART 2: Admin Schedule — GitHub Auto-Publish Setup
──────────────────────────────────────────────────────────

The admin page saves the schedule DIRECTLY to GitHub so
changes appear on the live site in about 60 seconds.
No downloads. No manual uploads. Just hit "Publish".

STEP 1 — Create a GitHub Personal Access Token
  → Go to https://github.com/settings/tokens/new
  → Note:       Rampagers Admin
  → Expiration: No expiration (or 1 year)
  → Scopes:     check "repo" (full repo access)
  → Click "Generate token"
  → COPY the token now (you only see it once!)

STEP 2 — Enter it in admin.html
  → Go to your site and open admin.html
  → Log in with password: Rampagers123!
  → Click the ⚙️ gear / settings icon
  → Enter your GitHub username, repo name, and token
  → Click "Save Settings"
  → The token is stored only in your browser's localStorage

STEP 3 — Publish!
  → Edit the schedule, click "🚀 Publish to Site"
  → GitHub commits the new runs.json automatically
  → The live site updates within ~60 seconds
