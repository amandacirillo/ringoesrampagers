⚡ One-time setup: Connect to Google Sheets (~5 min)

  1. Go to sheets.google.com → create a blank sheet named Rampagers Schedule
  2. Click Extensions → Apps Script
  3. Delete the code there and paste in the script from apps-script.txt (in your download)
  4. Click Deploy → New deployment → Type: Web app → Execute as: Me → Access: Anyone
  5. Click Deploy, authorize when prompted, then copy the Web App URL
  6. Paste it below and click Save
